üéØ What is Linear Regression?

Linear Regression is a fundamental machine learning algorithm that finds the best straight line to fit through data points. It's used to predict a continuous value (like house price) based on one or more input features (like house size).

üìê The Math Behind It

Linear regression finds a line described by the equation:

y = mx + b

Where:

  • y = predicted price (dependent variable)
  • x = house size (independent variable)
  • m = slope (how much price increases per sq ft)
  • b = intercept (base price when size is 0)

üéØ How Does It Learn?

The algorithm uses a method called Ordinary Least Squares (OLS) to find the optimal line. It minimizes the sum of squared differences between actual prices and predicted prices. This ensures the line is as close as possible to all data points.

üìä Understanding R¬≤ Score

R¬≤ (R-squared) measures how well the model explains the variance in the data:

  • R¬≤ = 1.0: Perfect fit (all points on the line)
  • R¬≤ = 0.96: Excellent fit (96% of variance explained)
  • R¬≤ = 0.0: Model is no better than guessing the mean
  • R¬≤ < 0.0: Model performs worse than the mean

‚öñÔ∏è Model Metrics Explained

RMSE (Root Mean Squared Error): Average prediction error in the same units as the target variable. Lower is better. Our model has an RMSE of ~$13.74K, meaning predictions are typically within $13,740 of the actual price.

MAE (Mean Absolute Error): Average absolute difference between predicted and actual values. More intuitive than RMSE as it's not squared.

üîç Limitations & Assumptions

Linear regression assumes:

  • A linear relationship between features and target
  • Independence of observations
  • Homoscedasticity (constant variance of errors)
  • Normally distributed residuals

In real estate, many factors affect price (location, age, condition, etc.), so this simple model is a starting point. More complex models would include multiple features.
